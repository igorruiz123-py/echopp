<h1 align="center">
    <i>
        <strong>echo++</strong>
    </i>
</h1>

<p align="center">
    <i><strong>echo++</strong> is a CLI open source software inspired originally on the traditional command "echo" that outputs whatever string the user inputs right after the "echo" command.</i>
</p>

<p align="center">
    <i>The idea of <strong>echo++</strong> is to extend its usage, by adding new features with <strong>flags.</strong> A flag, on this context, is a paramerater that is passed with the command and modifies the string according with the flag that is being used.</i>
</p>

<p>
    <i>for further information about <strong>flags</strong> see the following manual on how to use them and the meaning of each flag: <a href="https://github.com/igorruiz123-py/echopp/blob/main/docs/manual.md">manual.md</a>.</i>
</p>

<h2 align="center">
    <i>
        <strong>Installation</strong>
    </i>
</h2>

<p align="center">
    <i>
    A Makefile is a text file that contains instructions (rules, variables, and directives)
    used by the make utility to automate tasks, most commonly the compilation and linking of software projects. 
    </i>
</p>

<p align="center">
    <i>
    Makefile is used on <strong>echo++</strong> to compile the C source code into the executable "<strong>echo++</strong>" file and install it on the user's path.
